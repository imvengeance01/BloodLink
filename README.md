## ğŸ”„ System Overview 

BloodLink follows a simple and effective emergency workflow:

1. Donors register with blood group, location, and basic details
2. Patients or hospitals raise an emergency blood request
3. The system filters eligible donors based on blood group and cooldown period
4. Nearby donors are matched using location-based logic
5. Emergency alerts are sent to multiple donors simultaneously
6. The fastest available donor is selected based on live ETA
7. Direct coordination enables successful blood donation

This workflow minimizes manual effort and saves critical time.
